6 Key Improvements

01
Miller-Rabin Primality Test
Replaced the naïve O(√n) sieve with the probabilistic Miller-Rabin test running in O(k·log²n). Error probability < 4⁻²⁰ ≈ 10⁻¹².
Performance
02
Random Prime Generation
Keys are now drawn from a cryptographically random pool instead of always picking the largest prime below a fixed bound — a basic requirement for real-world RSA.
Security
03
e = 65537 Standard Exponent
The original iterated e starting from 2, producing dangerously small exponents. Fixed to Fermat prime F₄ = 65537, the industry standard for fast and safe modular exponentiation.
Security
04
modinv() Bug Fix
The original called extended_euclid(a, m) twice in modinv — once to check the gcd, once to retrieve x. Fixed to a single call with a proper ValueError on failure.
Bug Fix
05
Full UTF-8 Encoding
Replaced the hand-built 26-letter dictionary with standard UTF-8 byte encoding. Supports accents, uppercase, digits, punctuation, emoji, CJK — any valid Unicode character.
Feature
06
Byte-Aligned Block Splitting
Rewrote the fragile decimal-string splitting logic. Blocks are now fixed-size byte chunks of length byte_len(n) - 1, mathematically guaranteeing every block < n.

extended_euclid.py — iterative, single-call fix Python 
def extended_euclid(a: int, b: int) -> tuple:
    # Returns (d, x, y) s.t. a·x + b·y = d = gcd(a,b)
    x0, x1, y0, y1 = 0, 1, 1, 0
    while a != 0:
        q, a, b = b // a, b % a, a
        y0, y1 = y1, y0 - q * y1
        x0, x1 = x1, x0 - q * x1
    return b, x0, y0

def is_prime(n: int, rounds: int = 20) -> bool:
    # Write n-1 as 2^r · d
    r, d = 0, n - 1
    while d % 2 == 0:
        r += 1; d //= 2
    bases = random.sample(range(2, min(n-2, 1000)), min(rounds, n-4))
    return all(_miller_rabin_round(n, d, r, a) for a in bases)
encrypt.py — byte-aligned block splitting Python 
def encrypt(message: str, public_key: tuple) -> list:
    n, e = public_key
    block_size = (n.bit_length() + 7) // 8 - 1  # guarantees block < n
    raw    = message.encode("utf-8")
    padded = raw + b"\x00" * ((-len(raw)) % block_size)
    return [pow(int.from_bytes(padded[i:i+block_size], "big"), e, n)
            for i in range(0, len(padded), block_size)]
