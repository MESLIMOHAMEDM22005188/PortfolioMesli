Enterprise Cybersecurity Lab

Architecture d'entreprise complète intégrant Windows Server, Active Directory, Ubuntu Server et des outils offensifs Kali Linux dans un environnement de lab virtuel.

Architecture

L'environnement simule un réseau d'entreprise classique : un contrôleur de domaine Windows Server 2022, deux postes clients (Windows 11 et Windows 10), un serveur Ubuntu exposant Apache, et une machine Kali Linux pour les tests offensifs.

Machines

  • DC01 — Windows Server 2022 : Active Directory, DNS, DHCP, GPO
  • CLIENT01 — Windows 11, machine du domaine
  • CLIENT02 — Windows 10, machine du domaine
  • WEB01 — Ubuntu Server : Apache, SSH, Fail2Ban
  • KALI — Kali Linux : Nmap, Wireshark, Burp Suite, Hydra, Metasploit

Active Directory

  • 4 unités d'organisation (OU)
  • 30 utilisateurs créés
  • 5 groupes de sécurité

GPO mises en place

  • Politique de mot de passe complexe
  • Verrouillage du compte après 5 essais
  • Désactivation du panneau de configuration
  • Fond d'écran d'entreprise déployé

Configuration Linux

  • SSH par authentification par clé
  • Fail2Ban pour la protection contre le bruteforce
  • Tâches Cron planifiées
  • Sauvegardes automatiques via Rsync

Sécurité & Journalisation

Scan réseau effectué avec Nmap, analyse de paquets avec Wireshark. Journalisation côté Linux via journalctl, côté Windows via l'Event Viewer.

Scripts PowerShell

  • Création automatique d'utilisateurs
  • Scripts de sauvegarde
  • Génération de rapports
