Enterprise Cybersecurity Lab

A hands-on enterprise lab designed to simulate a real-world infrastructure combining Windows Server, Linux services and offensive security tools.

Overview

This project reproduces a small enterprise environment using Oracle VirtualBox. The objective is to develop practical skills in system administration, networking, Active Directory, Linux hardening and cybersecurity.
