Wazuh · Elasticsearch · Kibana

SOC + SIEM Lab

A Security Operations Center (SOC) laboratory designed to centralize logs, detect suspicious activities and provide real-time monitoring capabilities for Windows and Linux systems.

Overview

This project focuses on building a small Security Operations Center using Wazuh, Elasticsearch and Kibana. The objective is to collect logs from Windows and Linux systems, detect attacks and visualize security events through dashboards.
