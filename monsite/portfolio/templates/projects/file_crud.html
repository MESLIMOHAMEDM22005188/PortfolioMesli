Gestionnaire de fichiers – Application de bureau

Application conçue pour aider les personnels à gérer efficacement leurs fichiers et dossiers sur Windows. Une version mobile est en cours de développement.

Fonctionnalités clés

Ajout et suppression de fichiers

Les utilisateurs peuvent ajouter, supprimer ou déplacer leurs fichiers facilement.

Création de dossiers & catégories

Organisation intelligente des fichiers grâce à un système de dossiers et de catégories personnalisées.

Visualisation & typage

Affichage clair des types de fichiers et de leurs contenus, triables selon différents critères.

Développée en Python avec une architecture MVC claire, cette application de bureau fonctionne sous Windows. Elle a été conçue durant mon stage pour simplifier la gestion documentaire du personnel. Une version mobile est actuellement en cours de développement pour compléter l'expérience utilisateur.
