TaskFlow – Secure Task Manager

Application complète de gestion de tâches sécurisée avec authentification JWT, architecture modulaire et base de données relationnelle, conçue pour offrir une gestion fiable, scalable et sécurisée des workflows utilisateurs.

Fonctionnalités principales

Authentification sécurisée

Système d’authentification basé sur JWT avec gestion des sessions, protection des routes et sécurité des données utilisateur.

Gestion complète des tâches

Création, modification, suppression et suivi des tâches avec persistance en base MySQL et gestion multi-utilisateurs.

Architecture modulaire

Backend structuré avec Node.js, services, controllers et routes séparés pour garantir une maintenabilité et une scalabilité optimales.

Base de données relationnelle

Utilisation de Prisma ORM avec MySQL pour assurer la cohérence des données et simplifier les opérations CRUD.

Interface utilisateur moderne

Frontend React moderne avec gestion des états, appels API sécurisés et interface responsive.

Architecture sécurisée

Middleware d’authentification, validation des requêtes et séparation claire entre frontend et backend.

Stack technique

React Node.js Express Prisma ORM MySQL JWT REST API

TaskFlow est une application complète développée avec une architecture moderne frontend/backend séparée. Elle met en œuvre des concepts avancés d’ingénierie logicielle, notamment la sécurité JWT, l’architecture modulaire, l’ORM Prisma et la gestion de base de données relationnelle.
