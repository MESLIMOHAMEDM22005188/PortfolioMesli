Application de gestion de tickets – Projet web

Application web développée avec Django pour permettre aux utilisateurs de créer, suivre et gérer leurs tickets de support. L'interface est claire, moderne et intuitive.

Fonctionnalités principales

Création de tickets

Formulaire complet avec catégorie, niveau d’urgence, description détaillée, pièce jointe et date du problème.

Interface utilisateur

Accueil personnalisé, consultation des tickets créés, design responsive et agréable.

Panneau administrateur

L’administrateur peut consulter les pôles, créer des tickets, modifier les descriptions de pôles, gérer les utilisateurs et répondre aux alertes. Tout est centralisé dans un tableau de bord clair, en mode sombre.

Projet réalisé avec Python, Django, HTML/CSS et Tailwind. L’administration est pensée pour une gestion fluide des tickets et une modération par pôle. Le système peut être enrichi avec de nouveaux rôles, alertes et statistiques. Idéal pour les structures nécessitant un suivi clair et personnalisable.
