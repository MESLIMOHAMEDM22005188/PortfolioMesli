Application Java d’Analyse Sismique

Développée en Java avec JavaFX, cette application permet d’analyser et de visualiser des données sismiques issues du site SisFrance. Réalisée dans le cadre de la SAE 2.01 en une semaine, elle propose des filtres avancés, des graphiques interactifs et une carte dynamique.

Outils & Technologies

JavaFX & FXML
Framework d’interface pour application Java.
IntelliJ IDEA
IDE utilisé pour tout le projet Java.
Git & GitHub
Suivi de version et collaboration continue.
Overleaf (LaTeX)
Rédaction collaborative du rapport.
Figma
Maquettes interactives & mockups.
PlantUML
Diagrammes UML et architecture.

Fonctionnalités Implémentées

  • Filtrage & tri de séismes depuis un fichier CSV
  • Autocomplétion sur les zones de recherche (ID, nom)
  • Visualisation sur carte Gluon avec points interactifs
  • Affichage de graphiques (camemberts, lignes)
  • Navigation entre 3 pages principales (recherche, tableau, dashboard)

Architecture MVVM

Le modèle MVVM a été utilisé pour structurer le projet : séparation claire entre les données, l’interface graphique et la logique de présentation. Organisation par packages : datahandling, graphics, resources.
