À propos de ce fork

Cette version améliorée du dépôt original “Cybersecurity-Mini-Projects” inclut des fonctionnalités supplémentaires, surtout pour le projet de keylogger. Les améliorations comprennent :

  • Chiffrement sécurisé des logs
  • Avertissements légaux pour la conformité
  • Code mieux structuré et documenté
  • Limites de temps et envoi sécurisé par e-mail

Présentation des Projets

Projet Description Langage(s)
RSA Algorithm Implémentation complète de RSA depuis zéro Python
Keylogger (Éducatif) Keylogger amélioré avec fonctionnalités de sécurité Python
Network Sniffer Scanner des appareils locaux du réseau Python
Phishing Pages Simulation éducative de phishing HTML / PHP
Steganography Implémentation LSB de stéganographie Python
SSTI Exemple d’injection de template côté serveur Python / Flask
